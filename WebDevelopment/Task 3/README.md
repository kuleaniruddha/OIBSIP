# Task 3 - Temperature Converter 🌡️

## 📌 Description

This is a responsive **Temperature Converter Web App** created as part of the Oasis Infobyte Web Development Internship (OIBSIP).  
It allows users to convert temperatures between **Celsius, Fahrenheit, and Kelvin** using a simple and clean user interface.

The project is built using **HTML**, **CSS**, and **JavaScript**.

---

## ✨ Features

- 🌡️ Input validation for numerical temperature values
- 🔄 Supports conversion between Celsius, Fahrenheit, and Kelvin
- 🎨 Stylish and responsive design
- ⚡ Real-time temperature conversion on button click
- 📱 Works smoothly across desktop and mobile screens
